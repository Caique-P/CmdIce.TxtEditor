Esse é o nosso projeto

Nele é possivel definir um titulo e uma fonte via comandos, alguns deles são:

fonte grande, fonte cursiva [...] fonte enorme pfv. Todos os comandos devem ser seguidos de "pfv" para serem listados.

Também é possivel inserir uma imagem ou gif, copiando a imagem e colando aqui, ou por comandos:

fruta, paisagem, cat, luna, chico, [...] seguidos de pfv... Ou colar a imagem aqui com o ctrl + v:

É possivel calcular algo utilizando a calculadora, pelo atalho "=". Ira abrir uma janela de calculos, basta substituir os valores e clicar em calcular. Ex: 1*2=2.0.
Para salvar, abrir ou mudar a fonte de apenas uma linha, utilize os botões na barra superior. Todos os arquivos são salvos na pasta do projeto com o formato em ".html".
